# 青科小智

> 一个面向青岛科技大学教务场景的非官方微信小程序前端，用于课表与成绩查询。

## 项目简介

`青科小智` 是一个基于原生微信小程序开发的校园工具项目，当前主要覆盖以下能力：

- 学号密码登录
- 今日课表展示
- 周课表浏览与课程详情查看
- 成绩查询（有效成绩 / 原始成绩）
- 本地缓存登录态、课表和成绩数据

当前仓库仅包含小程序前端，不包含配套后端服务，也不直接提供对学校教务系统的通用适配能力。

## 项目现状

这个项目目前更适合作为：

- 微信小程序课程表 / 成绩查询前端示例
- 校园类小程序课程项目或二次开发基础模板
- 教务接口代理服务的前端演示层

以下入口已经预留，但尚未完整实现：

- 考试查询
- 校历查询
- 个人中心完善

## 技术栈

## 已实现功能

### 1. 登录

- 支持通过学号和密码登录
- 登录成功后将后端返回的 `cookie` 作为 `token` 写入本地缓存

### 2. 首页

- 展示快捷入口
- 展示今日课程列表

### 3. 课表

- 支持整周课程展示
- 支持切换周次
- 支持课程详情查看
- 支持课表本地缓存

### 4. 成绩

- 支持有效成绩查询
- 支持原始成绩查询
- 支持按学期切换查看

## 本地运行

### 1. 安装依赖

```bash
npm install
```

### 2. 导入微信开发者工具

将项目目录导入微信开发者工具。

### 3. 构建 npm

如果开发者工具未自动生成依赖，请在微信开发者工具中执行“工具 -> 构建 npm”。

### 4. 配置接口地址

修改 `config.js` 中的 `baseUrl`：

```js
export default {
  env,
  baseUrl: {
    develop: 'http://localhost:3000',
    production: 'https://your-api.example.com'
  }
}
```

### 5. 配置小程序 AppID

如需发布或真机调试，请按你的实际情况修改 `project.config.json` 中的 `appid`。

## 接口约定

当前前端默认依赖以下接口：

- `POST /login`
- `GET /courses`
- `GET /scores`
- `GET /raw-scores`

请求头说明：

- 登录后会在请求头中附带 `token`

建议后端统一返回如下结构：

```json
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```

其中：

- `code = 0` 表示成功
- `code = -1` 表示业务失败
- `code = 403` 表示登录失效

## 使用与二次开发说明

在当前版本中，部分内容仍然和具体学校场景强绑定：

- 学期开始日期与总周数写在页面逻辑中，需要按实际学期调整
- 首页与个人页的部分图片依赖服务端静态资源
- 后端接口字段结构默认按当前项目约定解析

如果你希望迁移到其他学校或其他教务系统，建议优先抽离：

- 学校名称与文案
- 学期配置
- 登录态字段
- 课程与成绩接口适配层
- 首页资源地址
